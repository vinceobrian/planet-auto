# PlanetAuto - Car Rental Platform

A modern car rental platform for Kenya, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Browse available cars with filters
- User authentication (sign up/login)
- Car booking system
- Responsive design
- M-Pesa integration ready

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- NextAuth.js
- Prisma ORM

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see .env.example)
4. Run the development server: `npm run dev`

## Deployment

This project is deployed on Vercel. 

Visit the live site: [https://planetauto.vercel.app](https://planetauto.vercel.app)
